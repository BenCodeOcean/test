# DevOps Skills

Skills for deployment, infrastructure management, and operations automation.

## Available Skills

### codeocean-deployment

Deploy, update, delete, and manage Code Ocean environments in AWS accounts using the covpc CLI tool.

**Use cases:**
- Creating new deployments for testing features
- Updating existing deployments after code changes
- Managing deployment lifecycle
- Feature development and testing in real environments

**Install:**
```bash
claude skill install git@github.com:your-org/your-skills.git/devops/codeocean-deployment
```
