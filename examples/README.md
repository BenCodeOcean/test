# Example Skills

Simple example skills for learning and testing Claude Code skill development.

## Available Skills

### hello-world

A simple skill that prints a colorful "Hello World" message using ANSI color codes.

**Use cases:**
- Testing skill installation
- Learning skill development basics
- Demonstrating colorful terminal output
- Quick verification that skills are working

**Install:**
```bash
claude skill install https://github.com/BenCodeOcean/test.git/examples/hello-world
```

## Install All Example Skills

```bash
claude skill install https://github.com/BenCodeOcean/test.git/examples
```
